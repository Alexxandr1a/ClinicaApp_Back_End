 <h1> 🏥 ClinicaApp - Sistema de Gestão Médica</h1>

 <h3>Sistema completo para gerenciamento de clínicas médicas, desenvolvido com foco em organização, produtividade e controle de atendimentos.</h3>
 
<br>

 <h2> 📌Sobre o Projeto</h2>

 <p>
  O ClinicaApp é uma aplicação web desenvolvida para otimizar a administração de clínicas médicas, centralizando o gerenciamento de pacientes, médicos, consultas e prontuários em um único sistema.
 <p>
  A plataforma foi projetada para oferecer uma experiência intuitiva e eficiente, permitindo acompanhamento em tempo real das principais informações da clínica por meio de um dashboard interativo e funcionalidades voltadas à gestão operacional.
 </p>
  <br>

  <h2>🚀 Funcionalidades</h2>
   <ul>
      <li>✅ Cadastro de pacientes</li>
      <li>✅ Cadastro de médicos</li>
      <li>✅ Agendamento de consultas</li>
      <li>✅ Controle de status das consultas
       <li>Pendentes</li>
       <li>Concluídas</li>
       <li>Canceladas</li>
      </li>
      <li>✅ Dashboard com métricas em tempo real</li>
      <li>✅ Gerenciamento de prontuários médicos</li>
   </ul>
  <br>

  <h2>🛠️Tecnologias Utilizadas</h2>

  <h2>Backend</h2>
  <ul>
    <li>Java</li>
    <li>Spring Boot</li>
  </ul>
  
  <h2>Banco de Dados</h2>
  <ul>
   <li>PostgreSQL</li>
  </ul>
  
  <h2>Dependências</h2>
  <ul>
    <li>Spring Data JPA</li>
    <li>Hibernate</li>
    <li>Spring Web</li>
    <li>Spring Security</li>
    <li>PostgreSQL Driver</li>
    <li>Jakarta Servlet API</li>
    <li>ModelMapper</li>
    <li>Lombok</li>
  </ul>
  
  <h2>Interface do Sistema</h2>
  <ul>
   <li>Dashboard</li>
   <li>Pacientes</li>
   <li>Médicos</li>
   <li>Agendamento</li>
   <li>Prontuário</li>
  </ul>

  <h2>📂 Arquitetura e Objetivos do Projeto</h2>

  <p>Este projeto foi desenvolvido com o objetivo de praticar e aplicar conceitos modernos de desenvolvimento backend utilizando o ecossistema Spring.</p>

   <h2>Principais objetivos:</h2>
  <ul>
   <li>Desenvolvimento de APIs REST com Spring Boot</li>
   <li>Aplicação de boas práticas de arquitetura back-end</li>
   <li>Integração com banco de dados relacional</li>
   <li>Estruturação de projetos escaláveis</li>
   <li>Organização em camadas</li>
   <li>Gerenciamento de entidades e relacionamentos com JPA/Hibernate</li>
   <li>Implementação de autenticação e segurança com Spring Security</li>
  </ul>
  <br>

|   Módulo    |              Descrição                        |
| ----------- | --------------------------------------------- |
| Dashboard   | Visualização de métricas e informações gerais |
| Pacientes   | Cadastro e gerenciamento de pacientes         |
| Médicos     | Controle e gerenciamento de médicos           |
| Consultas   | Agendamento e controle de atendimentos        |
| Prontuários | Registro e acompanhamento médico              |
